# tokyo_restaurants

A curated list of some of the best affordable restaurants in Tokyo

### Usage - production

Production

```
npm install
make build
go run server.go
```

Then visit [localhost:4567](http://localhost:4567)

Development

```
npm install
make run
```

Then visit [localhost:8080](http://localhost:8080) (webpack dev server)

### Dependencies

* go
* npm
* angular
* restangular
* webpack
